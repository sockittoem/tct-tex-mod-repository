1844c: The Widow's Son

by tex, /u/RouteVenus and /u/Atom_H3art_Mother

1844c is a short attempt at changing up The Campaign Trail formula by placing the player in the role of Mormon prophet Joseph Smith. While running his real-life presidential campaign, the player will also have to evade death and potentially risk withdrawing from the race.

Get the Mod

Through the following codes:

Smith

Code 1

Code 2
Select all and copy within each box containing the code you wish to copy. Then, paste into the corresponding boxes provided in the 'Other' section under the Mod Loader. Alternatively, on the old Campaign Trail site, paste Code 1 first into the Console, then paste Code 2 upon loading the game (note: plenty of functionality will be missing).
